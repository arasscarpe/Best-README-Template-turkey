## Proje Hakkında

[![Ürün Ekran Görüntüsü][product-screenshot]](https://example.com)

GitHub’da birçok harika README şablonu var; ancak ben ihtiyaçlarıma gerçekten uyan bir tane bulamadım, bu yüzden bu geliştirilmiş versiyonu oluşturdum. O kadar iyi bir README şablonu yapmak istiyorum ki bir daha hiçbir şeye ihtiyaç duymayasınız — bence bu o şablon.

Bunun sebepleri:
* Zamanını harika şeyler üretmeye odaklamalısın. Problem çözen ve başkalarına fayda sağlayan projeler yapmalısın
* Sıfırdan sürekli README yazmak gibi tekrar eden işleri yapmamalısın
* Hayatının geri kalanında DRY prensibini uygulamalısın :smile:

Elbette hiçbir şablon tüm projeler için uygun olamaz çünkü ihtiyaçlar değişebilir. Bu yüzden ileride daha fazlasını ekleyeceğim. Ayrıca bu repoyu fork’layarak pull request açabilir veya issue oluşturup öneride bulunabilirsin. Katkıda bulunan herkese teşekkürler!

### Ön Gereksinimler

Bu bölüm, yazılımı çalıştırmak için gerekenleri ve nasıl kurulacağını gösterir.
* npm
```sh
npm install npm@latest -g
